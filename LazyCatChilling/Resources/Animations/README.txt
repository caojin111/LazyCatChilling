如何添加GIF动画文件：

1. 将cat_working.gif和cat_resting.gif文件放入此文件夹中
2. 确保文件名与代码中使用的名称完全一致
3. 在Xcode中，选择"Add Files to 'LazyCatChilling'..."将这些文件添加到项目中
4. 确保添加时选择"Copy items if needed"和"Create groups"选项
5. 在Target Membership中勾选"LazyCatChilling"

注意：
- GIF文件应该质量适中，避免文件过大导致性能问题
- 推荐大小为300x300像素左右，以获得最佳显示效果
- 如果动画不显示，请检查文件名和资源引用是否一致
